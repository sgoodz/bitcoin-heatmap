## Getting Started

1. **Clone the repo:**

   ```sh
   git clone https://github.com/YOUR_USERNAME/bitcoin-heatmap.git
   cd bitcoin-heatmap
   ```

2. **Install dependencies:**

   ```sh
   npm install
   ```

3. **Run locally:**

   ```sh
   npm run dev
   ```

4. **Build for production:**
   ```sh
   npm run build
   ```
